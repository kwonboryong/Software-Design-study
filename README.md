## 예제로 배우는 소프트웨어 디자인 스터디 


### 목표
- 책에서 소개하는 디자인 패턴과 원칙들을 실제 코드로 구현해보고, 이를 이해하고 적용할 수 있는 능력을 기른다.
- 소프트웨어 설계 시 흔히 마주치는 문제들을 해결하는 방법을 학습한다.
- 코드 리뷰 및 토론을 통해 디자인 패턴을 실무에서 효과적으로 사용할 수 있는 방법을 논의한다.
